# Contributing

Thanks for your interest in Pandoc Online!

1. Fork the repository and create a branch from `main`.
2. Install dependencies: `pnpm install` and `pip install -r apps/worker/requirements.txt`.
3. Run checks: `pnpm typecheck`, `pnpm test`, `pytest apps/worker/tests`.
4. Open a pull request with a clear description and tests where applicable.
